DFINITY

無限のトラスト・マシン

DFINITYは、オープンバーチャルブロックチェーンコンピュターであり、ビジネスのEthereumエコシステムを拡張・重視するテクノロジーです。そのイノベーションには、Blockchain Nervous Systemを利用したオープンチェーンの管理システムをはじめ、パブリックチェーンにおいて直接コントラクトを成立できるスマートコントラクトを備えたプライベートチェーン、さらに、これまでのバーチャルマシンの20倍のスピードや無限のスケーラビリティへのロードマップを提供する第3世代のクリプトおよびプロトコルなどがあります。

PHI

分散型商業銀行業務および安定した通貨

今日の商業銀行システムは解決できない深刻な問題に直面しています。 それには、支店、スタッフ、インフラストラクチャ、コンプライアンス・プロセスにかかる膨大なコスト、大規模な救済措置が必要となるローン発生時の慢性的な判断ミス、社会的影響力があり、切実に求められている財務革新を妨げる独占的な傾向などがあります。 商業銀行を人のバリデータのアルゴリズムとネットワークを利用する自律的スマートコントラクトと完全に入れ替えて信頼性の高いローンを提供したり、その二次的効果として、不換紙幣のような安定した仮想通貨を生み出すことを想像してみてください。

Research

BLS Threshold Cryptography

DFINITY, especially Threshold Relay Chain consensus, extensively uses Stanford BLS signature as a cryptographic VRF (verifiable random function) to achieve incorruptible, unmanipulable and unpredictible randomness to select block maker and validator groups.

It's also used in PHI autonomous commercial banking system as a powerful way to randomly select validators for financial loans to prevent collusion.
